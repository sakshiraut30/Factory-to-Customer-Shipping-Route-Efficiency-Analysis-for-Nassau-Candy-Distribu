# 🚚 Factory-to-Customer Shipping Route Efficiency Analysis  
### Nassau Candy Distributor

---

## 📌 Project Overview
This project is a **data analytics dashboard** built using Streamlit to analyze and optimize factory-to-customer shipping routes for Nassau Candy Distributor. It helps identify inefficiencies in logistics, delays in delivery, and performance differences across routes, regions, and shipping modes.

---

## 🎯 Objectives
- Analyze shipping **Lead Time (Order Date → Ship Date)**
- Identify efficient and inefficient routes
- Compare shipping modes performance
- Detect bottlenecks in delivery system
- Provide interactive visual insights for decision-making

---

## 📊 Key Features
- 📦 KPI Dashboard (Average Lead Time, Delay Frequency, Efficiency Score, Route Volume)
- 🗺️ Route-level performance analysis
- 🚚 Ship Mode comparison
- 📈 Interactive charts (scatter, line, bar, histogram, box, violin)
- 🏆 Route performance leaderboard
- 📍 State-level bottleneck analysis
- 🎛️ Filters (date range, ship mode, lead time threshold)
- 📊 Clean dark-themed UI

---

## 🧠 Tech Stack
- Python
- Streamlit
